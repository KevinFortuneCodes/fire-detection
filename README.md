# Fire Detection

This project utilizes computer vision and machine learning to detect the presence of wildfires (fire and smoke) using the D-Fire dataset. Our approach compares baseline algorithms—including K-Nearest Neighbors, Decision Trees, and Random Forests using engineered statistical features against Convolutional Neural Networks (CNNs) designed for robust image classification. The goal is to develop a reliable model capable of distinguishing between fire, smoke, and neutral backgrounds under various lighting conditions.

**Repository Structure Note**
Please note that the active code for this project is currently distributed across three separate branches to facilitate parallel development:
* **`baseline_models`** (Baseline Models & Feature Engineering)
* **`image_preprocessing_and_object_detection`** (Data Preprocessing Pipeline and Object Detection Implementation)
* **`CNN`** (Convolutional Neural Network Experiments)

All components will be merged into this `main` branch in future updates. Please switch to the specific branches to view or run the respective code modules.
